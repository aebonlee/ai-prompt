## 1단계: PE 개론

### 1-1. 프롬프트 엔지니어링이란? (`/intro/what-is-prompt`)
**학습 목표**: 프롬프트의 개념과 중요성을 이해하고, LLM 작동 원리를 파악한다.

**구성 내용**:
1. 프롬프트 정의 - AI에게 보내는 입력 텍스트
2. 왜 중요한가? - 같은 AI라도 프롬프트에 따라 결과가 달라짐
3. LLM 작동 원리 - 토큰화 → 어텐션 → 생성 (3단계)
4. 프롬프트 vs 프로그래밍 비교 표
5. 좋은/나쁜 프롬프트 비교 예시
6. 5가지 핵심 원칙: 구체성, 맥락, 형식, 단계, 예시

**연습**: 나쁜 프롬프트를 좋은 프롬프트로 개선하기

### 1-2. AI 모델 이해하기 (`/intro/ai-models`)
**학습 목표**: 주요 AI 모델의 특성과 용도를 이해하고, 모델별 최적 프롬프트를 파악한다.

**구성 내용**:
1. LLM이란? - 대규모 언어 모델 개요
2. 주요 모델 비교 표: ChatGPT vs Claude vs Gemini (개발사, 특징, 강점, 컨텍스트, 추천용도)
3. 모델별 프롬프트 팁 + 실제 예시
4. 모델 선택 가이드: 작업 유형별 추천 모델

**연습**: 같은 프롬프트를 다른 모델에 시도하고 결과 비교

## 2단계: 학습하기

### 2-1. 프롬프트 구조와 패턴 (`/learn/structure`)
**학습 목표**: 효과적인 프롬프트를 구성하는 구조를 배우고, 핵심 패턴을 익힌다.

**구성 내용**:
1. RTF 패턴: Role(역할) + Task(작업) + Format(형식)
2. CRISPE 프레임워크:
   - Capacity: 역할/전문성
   - Request: 구체적 요청
   - Insight: 배경/맥락
   - Statement: 핵심 지시
   - Personality: 톤/스타일
   - Experiment: 변형/대안 요청
3. 핵심 패턴 4가지:
   - Step-by-step: "단계별로 설명해주세요"
   - Delimiter: 구분자로 영역 분리
   - Negative Prompt: 하지 말아야 할 것 명시
   - Output Format: 출력 형식 지정

**연습**: CRISPE로 프롬프트 작성 실습

### 2-2. Few-shot Learning (`/learn/few-shot`)
**학습 목표**: 예시를 통한 학습 기법을 이해하고, Zero/One/Few-shot을 구분하여 활용한다.

**구성 내용**:
1. Zero/One/Few-shot 비교 + 예시
2. 실전 예제 5개:
   - 감정 분석 (리뷰 → 긍정/부정/중립)
   - 이메일 분류 (문의/불만/제안)
   - 자연어 → SQL 변환
   - 톤 변환 (격식체 ↔ 비격식체)
   - React 컴포넌트 패턴 생성
3. 효과적인 Few-shot 팁

**연습**: 자신의 업무 데이터로 Few-shot 프롬프트 작성

### 2-3. Chain-of-Thought (`/learn/chain-of-thought`)
**학습 목표**: 사고 과정을 유도하는 프롬프팅으로 복잡한 문제 해결 능력을 향상시킨다.

**구성 내용**:
1. CoT 개념: 단계적 사고 유도 기법
2. Zero-shot CoT: "단계별로 생각해보세요"
3. 실전 예제 4개:
   - 수학 문제 풀이
   - 논리 추론
   - 코드 디버깅 (단계별 분석)
   - 비즈니스 의사결정

**연습**: 복잡한 문제를 CoT로 해결

### 2-4. 역할 부여 (`/learn/role-system`)
**학습 목표**: AI에게 전문 역할을 부여하여 특화된 응답을 얻는 기법을 배운다.

**구성 내용**:
1. 역할 부여의 원리와 효과
2. 실전 역할 5가지:
   - 교육자: 비전공자 대상 설명
   - 개발자: 코드 리뷰
   - 번역가: 문맥 살린 번역
   - 분석가: 데이터 인사이트
   - 카피라이터: 광고 문구
3. 시스템 프롬프트 템플릿 구조

**연습**: 자신만의 전문가 역할 설계

## 4단계: AI활용 Tip!

### AI 종류별

#### ChatGPT (`/ai-tips/chatgpt`)
- ChatGPT란? GPT 버전별 비교 (GPT-3.5, GPT-4, GPT-4o)
- 주요 기능: 텍스트 생성, 코드 작성, 번역, 데이터 분석
- 프롬프트 팁: 역할 부여, 구체적 지시, 단계적 요청, 예시 제공
- 주의사항: 할루시네이션, 개인정보, 저작권, 최신 정보 제한

#### Claude (`/ai-tips/claude`)
- Claude란? Anthropic의 AI, 200K 토큰 문맥
- 주요 기능: 긴 문서 분석, 정교한 코딩, 논리적 추론
- 프롬프트 팁: XML 태그 활용, 체계적 지시, 사고 과정 요청
- ChatGPT와의 차이점 비교

#### Gemini (`/ai-tips/gemini`)
- Gemini란? Google의 멀티모달 AI
- 주요 기능: Google 서비스 연동, 실시간 검색, 멀티모달
- 프롬프트 팁: Google 검색 연동, 이미지 분석 활용

#### Copilot (`/ai-tips/copilot`)
- GitHub Copilot란? AI 코딩 어시스턴트
- VS Code 설정, 단축키, Chat 명령어
- 활용법: 코드 자동완성, 테스트 생성, 코드 설명
